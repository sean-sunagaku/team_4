# 🔑 Google Maps APIキーの取得方法（詳細ガイド）

## ステップ1: Google Cloud Consoleにアクセス

1. ブラウザで https://console.cloud.google.com/ を開く
2. Googleアカウントでログイン（GmailアカウントでOK）

## ステップ2: プロジェクトを作成

1. 画面上部の「プロジェクトを選択」をクリック
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を入力（例：「DriBiddy」）
4. 「作成」をクリック
5. プロジェクトが作成されるまで数秒待つ

## ステップ3: 必要なAPIを有効化

### 3-1. Maps JavaScript APIを有効化

1. 左メニューから「APIとサービス」→「ライブラリ」をクリック
2. 検索バーに「Maps JavaScript API」と入力
3. 「Maps JavaScript API」をクリック
4. 「有効にする」ボタンをクリック

### 3-2. Geocoding APIを有効化

1. 左メニューから「APIとサービス」→「ライブラリ」に戻る
2. 検索バーに「Geocoding API」と入力
3. 「Geocoding API」をクリック
4. 「有効にする」ボタンをクリック

### 3-3. Directions APIを有効化

1. 左メニューから「APIとサービス」→「ライブラリ」に戻る
2. 検索バーに「Directions API」と入力
3. 「Directions API」をクリック
4. 「有効にする」ボタンをクリック

## ステップ4: APIキーを作成

1. 左メニューから「APIとサービス」→「認証情報」をクリック
2. 画面上部の「認証情報を作成」をクリック
3. 「APIキー」を選択
4. APIキーが作成され、ポップアップに表示されます
5. **このAPIキーをコピー**（後で使います）

⚠️ **重要**: APIキーは他人に見せないでください！

## ステップ5: APIキーの制限を設定（推奨）

セキュリティのため、APIキーの使用を制限することをお勧めします：

1. 作成したAPIキーをクリック
2. 「アプリケーションの制限」で「HTTPリファラー（ウェブサイト）」を選択
3. 「ウェブサイトの制限」に以下を追加：
   - `http://localhost:*`
   - `http://127.0.0.1:*`
4. 「APIの制限」で「特定のAPIに制限」を選択
5. 以下のAPIを選択：
   - Maps JavaScript API
   - Geocoding API
   - Directions API
6. 「保存」をクリック

## ステップ6: プロジェクトにAPIキーを設定

1. DriBiddyフォルダにある `.env` ファイルを開く（テキストエディタで）
2. 以下のように書き込む（`=`の後にコピーしたAPIキーを貼り付け）：

```
VITE_GOOGLE_MAPS_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

例：
```
VITE_GOOGLE_MAPS_API_KEY=AIzaSyAbCdEfGhIjKlMnOpQrStUvWxYz1234567
```

3. ファイルを保存

## ステップ7: 開発サーバーを再起動

1. ターミナルに戻る
2. `Ctrl + C` を押してサーバーを停止
3. 再度 `npm run dev` を実行
4. ブラウザをリロード（F5キー）

## ✅ 完了！

これでマップが表示されるはずです！

## ❓ トラブルシューティング

### 「このAPIキーは使用できません」と表示される場合
→ APIキーの制限設定を確認してください。開発中は「制限なし」に設定しても動作確認できます。

### 「請求が有効になっていません」と表示される場合
→ Google Cloud Consoleで請求アカウントを設定する必要があります（無料クレジットでも必要です）。

### マップが表示されない場合
→ ブラウザのコンソール（F12キー）でエラーメッセージを確認してください。
