# 🚀 Webアプリの起動手順（初心者向け）

## ステップ1: ターミナルを開く

1. Macの場合：`Command + スペース` を押して「ターミナル」と検索
2. ターミナルアプリを開く

## ステップ2: プロジェクトフォルダに移動

ターミナルに以下のコマンドをコピー&ペーストしてEnterキーを押してください：

```bash
cd /Users/yukawashouhei/Desktop/DriBiddy
```

## ステップ3: 必要なパッケージをインストール

以下のコマンドを実行してください（数分かかります）：

```bash
npm install
```

これで、ReactやGoogle Mapsなどの必要なライブラリがダウンロードされます。

## ステップ4: 開発サーバーを起動

以下のコマンドを実行してください：

```bash
npm run dev
```

成功すると、以下のようなメッセージが表示されます：
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## ステップ5: ブラウザで開く

1. ブラウザ（Chrome、Safari、Firefoxなど）を開く
2. アドレスバーに `http://localhost:5173` と入力してEnterキーを押す
3. または、ターミナルに表示されているURLをクリック

## ⚠️ 注意：Google Maps APIキーが必要です

アプリを開いてもマップが表示されない場合は、Google Maps APIキーを設定する必要があります。

### APIキーの取得方法（簡単版）

1. ブラウザで https://console.cloud.google.com/ を開く
2. Googleアカウントでログイン
3. 新しいプロジェクトを作成
4. 「APIとサービス」→「ライブラリ」をクリック
5. 以下のAPIを検索して有効化：
   - **Maps JavaScript API**
   - **Geocoding API**
   - **Directions API**
6. 「認証情報」→「認証情報を作成」→「APIキー」をクリック
7. 作成されたAPIキーをコピー

### APIキーの設定方法

1. DriBiddyフォルダに `.env` というファイルを作成（テキストエディタで）
2. 以下の内容を書き込んで保存：

```
VITE_GOOGLE_MAPS_API_KEY=ここにあなたのAPIキーを貼り付け
```

例：
```
VITE_GOOGLE_MAPS_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

3. 開発サーバーを再起動（ターミナルで `Ctrl + C` で停止してから、再度 `npm run dev` を実行）

## 🛑 サーバーを停止する方法

ターミナルで `Ctrl + C` を押すと、開発サーバーが停止します。

## ❓ よくある問題

### 「npm: command not found」と表示される場合
→ Node.jsがインストールされていません。https://nodejs.org/ からインストールしてください。

### 「ポート5173が既に使用されています」と表示される場合
→ 既にサーバーが起動しているか、別のアプリがポートを使用しています。`Ctrl + C`で停止してから再度起動してください。

### マップが表示されない場合
→ Google Maps APIキーが正しく設定されているか確認してください。ブラウザのコンソール（F12キー）でエラーメッセージを確認できます。
