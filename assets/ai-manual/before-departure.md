# 発車前

## チェックリスト

- 座席のセッティング・位置調整をする
  - ブレーキペダルを踏み込んだ時に膝が少し曲がる位置
  - ハンドルを持った時に肘が少し曲がる角度
- ミラーを調整する
  - ルームミラー：後方が見えるように
  - サイドミラー：車体が少し見える位置
- シートベルトを装着する
- エンジンをかける
  - AT車：ブレーキを踏んだ状態でスタート
  - MT車：クラッチを踏み、ニュートラル確認後にスタート
- 周辺の安全確認を行う
  - 左後方目視 → 左ミラー → ルームミラー → 右ミラー → 右後方目視
