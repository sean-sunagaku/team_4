# 車線変更の注意点

## 基本手順

1. 周囲の道路状況を確認
2. 車線変更の3秒前にウインカーを出す
3. 安全確認を行う
   - 前方を直視
   - ルームミラーで後方確認
   - ドアミラーで横を確認
   - 目視で死角を確認
4. 緩やかにハンドルを切って車線変更

## チェックリスト

- ウインカーを3秒前に出す
- ルームミラーで後方確認
- ドアミラーで斜め後方確認
- 目視で死角を確認（振り返って直接見る）
- 3秒かけてゆっくり車線変更
- 急ハンドルは厳禁

## 安全確認のポイント

- ミラーだけでは死角がある
- 必ず目視で横を確認する
- 左への車線変更は路肩をすり抜けるバイクに注意

## ミラーの距離感をつかむコツ

- 信号待ちの間にミラーを確認
- 映っている車がどれくらい後方にいるか想像
- 実際に目で見て答え合わせ

## 車線変更のコツ

- ルームミラーに後方の車の全体が映っていれば安全な距離
- 後ろの車と同じくらいの速度で移動する
- 車体が真横に並行移動するイメージ
- ハンドルは手首でゆっくり押すイメージ
